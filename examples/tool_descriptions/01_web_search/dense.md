Search the web for current or external information.

Use when: user asks about recent events, information outside training data, or verification of specific claims from authoritative sources.

Do NOT use for: basic facts you already know (arithmetic, well-known history, canonical technical terms).

Failure modes: rate-limited at 50 calls/hour/session. Over-broad queries return low-signal results — prefer specific keywords.
